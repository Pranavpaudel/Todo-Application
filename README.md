# Todo App

A Todo application built with React and Vite, featuring task management with filtering, editing, and priority marking.

## Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Mark tasks as important
- Filter tasks by All, Important, and Completed
- Keyboard support (Enter to add task)

## Tech Stack
- React
- JavaScript
- Vite
- CSS

## Getting Started

https://github.com/Pranavpaudel/Todo-Application
cd todo-app
npm install
npm run dev
